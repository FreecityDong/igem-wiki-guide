iGEM 网页开发文档集

📌 为什么要制作这组网页？

本网页文档集是为了帮助 iGEM 队伍成员，特别是网页开发新手，快速了解和掌握制作 iGEM Wiki 网站的基本流程、内容规范、页面设计与部署方法。

iGEM 的官方 Wiki 页面是项目展示的重要窗口，不仅承载了项目的结构化内容、实验成果与成员介绍，也是获得评审关注和奖项的重要依据。因此，我们希望通过这组网页资料：

  • 统一队伍的页面开发规范
  • 提供可复用的网页结构模板
  • 降低非技术队员参与门槛
  • 实现内容与结构的分离，便于协作

📘 如何使用这组文档？

点击顶部导航栏进入各类文档页面，每一页均聚焦一个主题：

  • 设计与布局： 网页结构、布局建议与 sidebar 样式
  • 语法手册： HTML / CSS / JavaScript 快速参考
  • 开发说明： VS Code 使用、GitHub Pages 部署方法
  • 项目结构： HTML 方案与 React 架构示例

每个页面使用统一风格，并通过顶部导航栏进行快速跳转。你也可以将页面保存本地或部署到 GitHub Pages 上作为团队内部学习网站。
