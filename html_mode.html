一、项目概览

本项目采用 Python 后端与 HTML + Bootstrap 前端，适用于 iGEM 或科研展示型静态网站构建。

二、主要特性

  • 后端使用 Python（推测为 Flask）进行路由与页面渲染
  • 页面采用模块化 HTML 模板组合（layout + menu + footer）
  • 使用 Bootstrap 提供响应式布局与界面样式
  • 页面内容分布明确，覆盖 iGEM 核心模块
  • 支持 GitLab CI 自动部署

三、目录结构说明

1. 根目录文件

  • app.py：后端主程序入口（如 Flask 应用）
  • dependencies.txt：Python 依赖列表，供安装环境使用
  • .gitignore：Git 忽略上传的文件规则
  • .gitlab-ci.yml：GitLab 自动部署配置
  • README.md：项目说明文档
  • LICENSE：开源许可证声明

2. 前端资源 static/

  • bootstrap.bundle.min.js：Bootstrap 的 JS 功能组件
  • bootstrap.min.css：Bootstrap 样式库
  • style.css：自定义样式覆盖与页面调整

3. 页面模板 wiki/

  • layout.html：页面基础结构模板
  • menu.html：页面导航菜单模板
  • footer.html：通用页脚模板

四、本地部署方法

# 1. 安装依赖（以 Flask 为例）
$ pip install -r dependencies.txt

# 2. 启动应用（如果是 Flask）
$ python app.py

# 3. 访问地址
http://localhost:5000 或具体配置地址

五、适用场景

  • iGEM 项目展示网站
  • 高校实验室网站或研究课题介绍
  • 响应式静态内容组合页面项目
