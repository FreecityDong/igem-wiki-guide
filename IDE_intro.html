1. 安装 Visual Studio Code

  1. 访问官网下载安装 VS Code：https://code.visualstudio.com
  2. 建议同时安装 Git：https://git-scm.com

2. 安装推荐插件

  • Live Server： 实时预览 HTML 页面 ✅
  • HTML Preview： 本地预览 HTML 页面
  • Prettier： 自动格式化 HTML / CSS / JS
  • GitLens： 管理 Git 提交记录（可选）

点击左侧插件图标，搜索插件名称并安装。

3. 创建项目文件夹

  1. 新建文件夹，例如：igem-wiki
  2. 使用 VS Code 打开
  3. 创建结构如下：
igem-wiki/
├── index.html
├── style.css
└── script.js

4. 编写与预览网页

  1. 编写 index.html：
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>iGEM 项目主页</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>欢迎访问我们的项目！</h1>
  <script src="script.js"></script>
</body>
</html>
  1. 右键 index.html → Open with Live Server
  2. 浏览器中实时预览 HTML 页面

5. 使用 GitHub Pages 部署网站

步骤一：新建 GitHub 仓库

  1. 访问 github.com，创建仓库（如 igem-wiki）

步骤二：上传代码

git init
git remote add origin https://github.com/你的用户名/igem-wiki.git
git add .
git commit -m "首次提交"
git push -u origin master

步骤三：启用 GitHub Pages

  1. 进入仓库页面 → Settings
  2. 选择 Pages
  3. 选择分支 main 或 master，路径 /root
  4. 保存后等待几分钟
  5. 访问：
    https://你的用户名.github.io/igem-wiki

6. 注意事项

  • 确保 index.html 命名正确
  • 所有路径使用相对路径
  • 添加空文件 .nojekyll，避免 GitHub Pages 忽略某些资源
