一、网页三要素：HTML + CSS + JS

一个网页由三种技术组成：

  • HTML： 负责结构，决定网页显示的内容
  • CSS： 负责样式，美化网页，控制颜色、布局、字体等
  • JavaScript： 负责行为，让网页动起来

二、基础页面结构

<!DOCTYPE html>
<html lang="zh">
  <head>...</head>
  <body>
    <header>头部</header>
    <nav>导航栏</nav>
    <main>主要内容</main>
    <footer>页脚</footer>
  </body>
</html>

三、Sidebar（侧边栏）布局

Sidebar 常用于页面导航、目录显示、快速跳转等，常位于左侧或右侧。

<div class="container">
  <aside>侧边栏</aside>
  <main>主体内容</main>
</div>

.container {
  display: flex;
}
aside {
  width: 220px;
  background: #f0f0f0;
  padding: 1em;
}
main {
  flex: 1;
  padding: 1em;
}

四、Nav Bar（导航栏）结构与样式

导航栏（Nav Bar）通常位于网页顶部，用于展示页面链接。

<nav>
  <a href="index.html">首页</a>
  <a href="team.html">团队介绍</a>
  <a href="project.html">项目内容</a>
</nav>

nav {
  background: #34495e;
  padding: 1em;
}
nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

Nav Bar 与 Sidebar 可组合使用：顶部用于模块跳转，侧边用于章节导航。

五、实践任务

  1. 任务 1： 创建包含 header、nav、main、footer 的页面。
  2. 任务 2： 添加一个 Sidebar，列出“项目介绍 / 实验流程 / 团队成员”等。
  3. 任务 3： 使用 Flex 实现左右布局结构。
  4. 任务 4： 用 CSS 美化 Nav Bar，并实现鼠标悬停效果。
  5. 挑战任务： 模仿某个 iGEM Wiki 首页，完成一个仿真页面。
