1. 什么是 CSS？

CSS（层叠样式表）用于控制网页的视觉表现，如布局、颜色、字体、动画等。

2. 基本语法

选择器 {
  属性名: 属性值;
}

例如：

p {
  color: blue;
  font-size: 16px;
}

3. 常见选择器

  • h1：元素选择器
  • .box：类选择器
  • #main：ID选择器
  • div p：后代选择器
  • div > p：子选择器

4. 如何引入 CSS

方法一：行内样式

<p style="color: red;">我是红色文字</p>

方法二：内部样式表

<head>
  <style>
    p { color: green; }
  </style>
</head>

方法三：外部 CSS 文件

<link rel="stylesheet" href="style.css">

将样式写入 style.css 文件中，适合团队协作开发与页面复用。

5. 常用属性分类

  • 颜色与文本： color, font-size, text-align
  • 尺寸与边距： width, height, margin, padding
  • 边框与圆角： border, border-radius
  • 背景： background-color, background-image
  • 布局： display, position, flex, grid

6. 实用样式案例

卡片式信息块

.card {
  background: #ffffff;
  border: 1px solid #ccc;
  padding: 20px;
  border-radius: 8px;
  width: 300px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

按钮样式 + 悬停动画

.btn {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 6px;
  cursor: pointer;
  transition: background-color 0.3s ease;
}
.btn:hover {
  background-color: #2980b9;
}

导航栏水平菜单

nav a {
  text-decoration: none;
  padding: 0.5em 1em;
  display: inline-block;
  color: white;
  background-color: #2c3e50;
  margin-right: 1rem;
  border-radius: 4px;
}

7. Flex 布局基础

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
  • justify-content 控制主轴对齐方式
  • align-items 控制交叉轴对齐方式

8. 响应式设计示例

@media screen and (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

这段代码让容器在屏幕宽度小于600px时变为垂直排列。

9. 动画与过渡

.fade-in {
  opacity: 0;
  animation: fadeIn 1s forwards;
}
@keyframes fadeIn {
  to { opacity: 1; }
}
