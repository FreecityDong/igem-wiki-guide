1. HTML 文件基本结构

<!DOCTYPE html>
<html lang="zh">
  <head>
    <meta charset="UTF-8">
    <title>页面标题</title>
  </head>
  <body>
    页面内容
  </body>
</html>

2. 常见标签分类

  • 文本结构： <h1> ~ <h6>, <p>, <br>
  • 超链接与图片： <a>, <img>
  • 列表： <ul>, <ol>, <li>
  • 表格： <table>, <tr>, <td>, <th>
  • 布局结构： <header>, <nav>, <main>, <footer>, <div>, <section>, <aside>
  • 表单： <form>, <input>, <textarea>, <button>

3. 常用标签示例

标题与段落

<h1>主标题</h1>
<p>这是一个段落。</p>

超链接

<a href="https://igem.org">访问iGEM官网</a>

图片

<img src="team.jpg" alt="团队合照" width="300">

无序与有序列表

<ul>
  <li>项目背景</li>
  <li>实验方法</li>
</ul>

<ol>
  <li>第一步</li>
  <li>第二步</li>
</ol>

表格

<table>
  <tr><th>名称</th><th>分数</th></tr>
  <tr><td>队员A</td><td>95</td></tr>
</table>

输入表单

<form>
  姓名：<input type="text" name="name"><br>
  留言：<textarea></textarea><br>
  <button type="submit">提交</button>
</form>

4. 特殊字符

  • &lt; 表示小于号 <
  • &gt; 表示大于号 >
  • &amp; 表示和号 &
  • &nbsp; 表示空格

6. 快速练习建议

  1. 尝试使用 <h1> 到 <h6> 标签写一个章节标题结构。
  2. 创建一个页面，包含一张图片、一个链接和一段介绍文字。
  3. 写一个小表格，展示三位队员的名字和分工。
  4. 尝试创建一个表单，收集用户姓名和反馈。
