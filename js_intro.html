1. JavaScript 是什么？

JavaScript 是网页的行为层，配合 HTML（结构）和 CSS（样式），可实现交互、动态内容、动画等功能。

2. 如何引入 JavaScript

方式一：内联

<button onclick="alert('你好！')">点击我</button>

方式二：页面内部脚本

<script>
  console.log("欢迎来到 iGEM Wiki！");
</script>

方式三：外部脚本

<script src="main.js"></script>

推荐使用外部文件，有助于代码管理与复用。

3. 变量定义

let name = "iGEM";
const year = 2025;
var count = 0;
  • let：块作用域变量（推荐）
  • const：常量，不可修改
  • var：函数作用域变量（已不推荐）

4. 常见数据类型

  • 字符串："Hello", 'World'
  • 数字：42, 3.14
  • 布尔值：true, false
  • 数组：[1, 2, 3]
  • 对象：{name: "Alice", age: 20}
  • 空值：null, undefined

5. 条件判断

let score = 90;
if (score >= 90) {
  console.log("优秀");
} else {
  console.log("加油！");
}

6. 循环语句

for (let i = 0; i < 5; i++) {
  console.log("第 " + i + " 次");
}

7. 函数定义与调用

function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("iGEM"));

8. 事件监听

document.getElementById("btn").addEventListener("click", function() {
  alert("按钮被点击了！");
});

对应 HTML 示例：

<button id="btn">点击按钮</button>

9. DOM 操作（网页元素控制）

// 修改元素文字
document.getElementById("title").innerText = "新标题";

// 改变样式
document.querySelector(".box").style.backgroundColor = "lightblue";

10. 实用案例

显示当前时间

<p id="time"></p>
<script>
  const now = new Date();
  document.getElementById("time").innerText = now.toLocaleString();
</script>

切换颜色按钮

<button onclick="toggleColor()">切换颜色</button>
<div id="box" style="width:100px;height:100px;background:red;"></div>

<script>
  function toggleColor() {
    let box = document.getElementById("box");
    box.style.background = (box.style.background === "red") ? "green" : "red";
  }
</script>
